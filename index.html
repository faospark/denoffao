FFVI ArtOmnia (AO)

Opera Omnia Style Character Portraits Version 3.0

Description :
A collection of Character Portraits that convey a 2d hand drawn chibi art style. The main goal of this mod is to essentially have a consistent cartoon like aesthetic of the game which is somewhat taken away by the almost 3d like inking of the Amano Art portraits. This mod is heavily inspired and features Official Dissidia Final Fantasy Opera Omnia (DFFOO). To compensate for characters that lack official artwork from DFFOO, a set of art has been curated to somewhat match the chibi art style of DFFOO. the fascinating thing is that most of the art featured pre-dates the creation of Opera Omnia. This mod is NOT released for commercial purposes
